# Play Store screenshots checklist

Capture on a **production or preview build** (not Expo Go), portrait phone, min 2 screenshots (recommend 4–8).

| # | Screen | What to show |
|---|--------|----------------|
| 1 | Home | Logo, tagline, Shop + Portal buttons, language chip (EN/HU/RO) |
| 2 | Shop WebView | muulox.com store loaded, home/reload controls visible |
| 3 | Portal WebView | Client portal logged in or login page |
| 4 | Language | Home with RO or HU UI (optional second locale shot) |
| 5 | Error state (optional) | Airplane mode → retry UI |

## Feature graphic

- Size: **1024 × 500** px  
- Save as: `assets/store/feature-graphic.png` (create in design tool; not generated by repo)

## Store icon

- Use `assets/AppIcons/playstore.png` (512×512) — verify sharp edges on device

## Before upload

- [ ] CMS legal pages live (`/aplicatie-mobila/...`)
- [ ] Privacy link in app opens published URL
- [ ] Cold start: splash shows brand background `#F3FBFC` and logo
- [ ] `eas build --profile preview --platform android` tested on device
