# MUULOX — descriere completă Play Store (RO)

Aplicația oficială MUULOX oferă acces rapid la:

**Magazin online** — Răsfoiți și comandați mii de produse B2B pentru retail și HoReCa pe muulox.com.

**Portal clienți** — Autentificare în panoul companiei, documente și servicii într-un singur loc.

**Limbi** — Aplicația este disponibilă în română, maghiară și engleză.

Aplicația este un wrapper securizat pentru serviciile web MUULOX: folosiți aceleași conturi, prețuri și checkout ca pe site. Comenzile, livrarea și garanțiile sunt guvernate de termenii site-ului MUULOX.
